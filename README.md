# UnityTeachDemo - 塔防项目练习

基于 Unity 3D 开发的塔防类游戏练习项目。

## 项目简介

本项目为 Unity 塔防类游戏 Demo，主要用于学习与实践游戏系统架构设计与核心玩法逻辑。
项目实现了塔防游戏的基础玩法流程，包括：
- 敌人生成与路径移动
- 防御塔攻击逻辑
- 伤害计算系统
- 波次管理
- UI 信息显示
- 游戏胜负判定

## 技术栈

- Unity 3D
- C#
- NavMesh / 路径移动
- 面向对象设计
- 基础 UI 系统

## 核心功能模块

### 敌人系统
- 敌人生成管理
- 固定路径移动
- 生命值管理
- 死亡处理逻辑

### 防御塔系统
- 自动索敌
- 攻击冷却控制
- 子弹生成与伤害判定
- 攻击范围检测

### 波次管理系统
- 敌人波次控制
- 间隔生成机制
- 游戏进度控制

### UI 系统
- 血量显示
- 波次信息展示
- 游戏状态提示

## 学习目标

通过本项目重点学习：
- 游戏核心玩法逻辑拆分
- 系统之间的调用关系
- 管理类（Manager）的组织方式
- 面向对象编程在游戏中的应用
